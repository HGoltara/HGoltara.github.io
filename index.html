Sobre

Aprendendo construindo produtos reais

Meus projetos mais recentes giram em torno do mesmo objetivo: sair do zero e chegar a um sistema funcional, com autenticação, persistência de dados, testes automatizados e documentação — não apenas telas bonitas. Passei por duas aplicações completas nesse processo: uma API de tarefas com autenticação e interface web, e um encurtador de URLs com documentação interativa. Em ambas, revisei e testei cada parte do código antes de considerá-lo pronto, e usei o Claude Code como apoio de produtividade — não como piloto automático.
